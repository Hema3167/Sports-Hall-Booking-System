# Sports Hall Booking System

## Project Overview
The Sports Hall Booking System is a ServiceNow-based web application developed to simplify the process of finding and booking sports venues. The system provides a user-friendly Sports Zone portal where users can explore available sports halls, make bookings, view their bookings, and check booking status.

## Features
- View available sports venues
- Book sports halls
- View personal bookings
- Check booking status
- Store and manage booking records
- Generate booking reports and analytics
- Responsive Sports Zone portal interface

## Technologies Used
- ServiceNow
- Service Portal
- HTML
- CSS
- JavaScript
- ServiceNow Tables and Forms
- ServiceNow Reports and Analytics

## Main Modules
1. Sports Zone Home Page
2. Book Venue
3. My Bookings
4. Booking Status
5. Sports Hall Booking Management
6. Reports and Analytics

## Project Output
The application provides a centralized platform for managing sports hall bookings and allows users to access venue information and booking services through a simple web portal.
